# Node.js와 AWS RDS (MySQL)을 활용한 웹개발프로젝트
## 개발배경
- 중고 거래를 도와주는 플랫폼은 시중에 많이 있었지만, 대부분 모바일 환경에서만 작동한다. PC에서도 간편하게 중고 거래를 할 수 있도록 도와주는 환경이 있으면 좋겠다는 생각으로 개발하게 되었다.
